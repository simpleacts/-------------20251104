# ページネーション実装状況

## 使用場所

✅ **商品選択モーダルの商品カード表示部分**で使用されています。

```typescript
// src/features/estimator/organisms/ProductSelectionGrid.tsx
<ProductSelectionGrid
    data={productsForDisplay}
    paginationConfig={{ enabled: true, itemsPerPage: 24 }}
    // ...
/>
```

`ProductSelectionGrid`内で`PaginatedListContainer`を使用して、商品カードのページネーションを実装しています。

## 実装されている機能

### ✅ 実装済み

1. **最初のページへ**（`<<`）ボタン
   - 1ページ目に移動
   - 1ページ目の場合は無効化

2. **前のページへ**（`<`）ボタン
   - 前のページに移動
   - 1ページ目の場合は無効化

3. **次のページへ**（`>`）ボタン
   - 次のページに移動
   - 最後のページの場合は無効化

4. **最後のページへ**（`>>`）ボタン
   - 最後のページに移動
   - 最後のページの場合は無効化

5. **現在のページ / 総ページ数**の表示
   - 例: `2 / 5`

6. **表示件数**の表示
   - 例: `25 - 48 / 120件`

7. **ページの自動リセット**
   - メーカー切り替え時に1ページ目にリセット
   - 検索・フィルタリング時に1ページ目にリセット

### ❌ 未実装

1. **ページ番号を直接選択する機能**
   - ドロップダウンでページ番号を選択
   - 入力フィールドでページ番号を入力
   - 現在は「前へ」「次へ」ボタンでのみ移動可能

## ページネーションコントロールの表示位置

- **ヘッダー**: ページネーションコントロールを表示
- **フッター**: ページネーションコントロールを表示
- **設定**: `paginationPosition="both"`

## 動作確認

### 基本的な動作

1. **1ページあたり24件表示**
2. **総ページ数が1ページ以下の場合はページネーションコントロールを非表示**
3. **データが変更されると自動的に1ページ目にリセット**

### メーカー切り替え時

1. メーカーを切り替える
2. データが再読み込みされる
3. **自動的に1ページ目にリセット**
4. 新しいメーカーの商品が1ページ目から表示される

### 検索・フィルタリング時

1. 検索語を入力、またはフィルタを変更
2. `filteredData`が再計算される
3. **自動的に1ページ目にリセット**
4. フィルタリングされた商品が1ページ目から表示される

## まとめ

- ✅ 商品選択モーダルの商品カード表示部分で使用されている
- ✅ 基本的なナビゲーションボタン（最初、前、次、最後）は実装済み
- ✅ 現在のページ / 総ページ数の表示は実装済み
- ✅ 表示件数の表示は実装済み
- ❌ ページ番号を直接選択する機能は未実装

基本的なページネーション機能は実装されていますが、ページ番号を直接選択する機能は未実装です。必要であれば、ドロップダウンや入力フィールドを追加できます。
