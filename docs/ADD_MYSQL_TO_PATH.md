# MySQL/MariaDBをPATHに追加する手順

## 見つかったパス

`C:\Program Files\MariaDB 12.0\bin`

このパスをシステム環境変数のPathに追加します。

## 手順

### ステップ1: システムの詳細設定を開く

1. **Windowsキー + X** を押す
2. 「**システム**」を選択
   - または、設定 → システム → 詳細情報

### ステップ2: 環境変数を開く

1. 「**システムの詳細設定**」をクリック
2. 「**環境変数**」ボタンをクリック

### ステップ3: Pathを編集

1. 「**システム環境変数**」セクションで「**Path**」を選択
2. 「**編集**」ボタンをクリック

### ステップ4: 新しいパスを追加

1. 「**新規**」ボタンをクリック
2. 以下のパスを入力：
   ```
   C:\Program Files\MariaDB 12.0\bin
   ```
3. 「**OK**」をクリック

### ステップ5: すべてのダイアログを閉じる

1. 「環境変数」ダイアログで「**OK**」
2. 「システムのプロパティ」ダイアログで「**OK**」

### ステップ6: 新しいPowerShell/コマンドプロンプトを開く

**重要**: 既存のウィンドウは閉じて、**新しいウィンドウ**を開いてください。
これにより、新しいPATH設定が読み込まれます。

### ステップ7: 確認

新しいPowerShellまたはコマンドプロンプトで以下を実行：

```bash
mysql --version
```

または

```bash
where mysql
```

以下のように表示されれば成功です：

```
C:\Program Files\MariaDB 12.0\bin\mysql.exe
```

## 接続テスト

PATH設定が完了したら、以下のコマンドで接続できます：

```bash
mysql -u root -p
```

パスワードを入力して接続します。

## トラブルシューティング

### まだ認識されない場合

1. **新しいウィンドウを開いたか確認**
   - 既存のウィンドウでは設定が反映されません
   - 必ず新しいPowerShell/コマンドプロンプトを開いてください

2. **パスが正しいか確認**
   - エクスプローラーで `C:\Program Files\MariaDB 12.0\bin\mysql.exe` が存在するか確認

3. **環境変数の再読み込み**
   - 一度ログアウトして再ログイン
   - または、PCを再起動

### 一時的な解決策

PATH設定が完了するまで、完全パスで実行できます：

```bash
"C:\Program Files\MariaDB 12.0\bin\mysql.exe" -u root -p
```
